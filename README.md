## Student Daily Necessities Procurement System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3c6e4619aaae4c509acff8979fadbf06/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Products
5. Product Information
6. Personal Center
7. Delivery Address
8. Product Collection
9. Product Orders
10. Shopping Cart


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management 
4. Basic Data Management (product type management, information type management)
5. Product Management
6. Product Review Management
7. Product Collection Management
8. Product Order Management
9. Product Information Management
10. User Management
11. Carousel Image Information
